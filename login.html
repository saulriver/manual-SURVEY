2. Login

Para poder ingresar a la aplicación Encuesta Software se requiere que el usuario se identifique con su nombre de usuario y la clave correspondiente.
Al ingresar el usuario y la contraseña también se puede seleccionar el lenguaje que el usuario crea conveniente para su proceso en el aplicativo inglés o español, le damos clic en Login para poder ingresar al aplicativo y si queremos cancelar el proceso se escoge la opción cancelar.
Si al ingresar las credenciales no estén asignadas ya sea un usuario que no existe o una contraseña, mostrará un mensaje de información de usuario o clave no valida verifique por favor.
Ingresamos las credenciales asignadas damos clic en Login validará la información y permitirá el ingreso al aplicativo.
Posteriormente a la autenticación exitosa, el sistema automáticamente dirige al usuario a la pantalla inicial del aplicativo Software Encuesta.
